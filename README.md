# Odin Recipes

A practice project for HTML basic fundamentals.

## Project Structure

```
odin-recipes/
├─ index.html           # Main page with links to recipes
├─ pages/               # Folder for recipe pages
│  ├─ recipe-name/
│  │  ├─ recipe-name.html
│  │  └─ assets/        # Images for the recipe
└─ README.md
```

## Getting Started

1. Open `index.html` in your web browser to view the site
2. Each recipe page should include:
   - A recipe title
   - An image of the finished dish
   - A description of the recipe
   - An ingredients list
   - Step-by-step instructions
